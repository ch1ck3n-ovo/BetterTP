# BetterTP

**Version** `1.20.4`

+ /btp help:

+ /btp list:

+ /btp remove ***name***: 
  + Permission: **btp.remove**
+ /btp set ***name***: 
  + Permission: **btp.set**
+ /btp tp ***name***: 
  + Permission: **btp.tp**
